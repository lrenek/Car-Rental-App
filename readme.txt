ΟΔΗΓΙΕΣ ΕΓΚΑΤΑΣΤΑΣΗΣ 

Για την αποτελεσματική λειτουργία του προγράμματος απαιτείται η εγκατάσταση της python. 
Είναι ακόμα απαιραίτητο οι φωτογραφίες caricon και car3 να βρίσκονται στον ίδιο φάκελο με τις εφαρμογές python.

Εγκατάσταση Python:
1.Από την επίσημη ιστοσελίδα της Python (https://www.python.org/) επιλέξτε και κατεβάστε το κατάλληλο αρχείο, σύμφωνα με τις προδιαγραφές του υπολογιστή σας.
2.Τρέξτε τον installer. Φροντίστε να επιλέξετε τα κουτάκια στα οποία αναγράφεται “Install Launcher for All Users” και “Add Python to PATH”.

Εγκατάσταση βιβλιοθηκών:
Numpy:
Για την εγκατάσταση πατάμε στο command prompt την εντολή : pip install –upgrade numpy.
Matplotlib:
Για την εγκατάσταση πατάμε στο command prompt την εντολή : pip install –upgrade matplotlib.

Εφαρμογές:

Βήμα πρώτο: Τρέξτε το αρχείο database.py 
Διαδικασία: Δεξί κλικ πάνω στο αρχείο->edit with idle->F5. 
To αρχείο αυτό χρειάζεται να το τρέξετε μόνο μία φορά.

Βήμα δεύτερο: Τρέξτε τις εφαρμογές client_app.py και admin_app.py, ακολουθώντας την παραπάνω διαδικασία.
